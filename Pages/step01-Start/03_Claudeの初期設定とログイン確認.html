// 最初の画面の見方

ログイン直後に表示されるClaude画面はシンプルです。大きく分けて「左側のサイドバー」と「右側のチャットエリア」の2つだけです。

💡
画面の構成はシンプルなので迷う場所はありません。一番よく使うのは右下のテキスト入力欄です。ここに話しかけるだけでClaudeが答えてくれます。
// 今すぐ確認しておくこと
3 ITEMS
👤
ログインできているか確認する
必須
画面左下（またはサイドバー下部）に自分のメールアドレスまたは名前が表示されていればログイン済みです。表示されていない場合は「Log in」をクリックしてログインしてください。
🌐
表示言語を確認する
確認推奨
Claudeは話しかけた言語で答えてくれます。日本語で入力すれば日本語で返してくれるので、言語の設定を変える必要は基本的にありません。英語で表示されていても、日本語で話しかければ日本語で返ってきます。
🔔
メール通知の設定（任意）
後でOK
Anthropicからのお知らせメールが届く設定になっています。不要な場合は「Settings（設定）」→「Notifications（通知）」からオフにできます。急がなければ後回しで問題ありません。
// 2回目以降のログイン方法
1
ブラウザで claude.ai を開く
ブックマーク（お気に入り）に登録しておくと毎回入力しなくて済みます。今のうちに登録しておくことをおすすめします。
💡
Chromeでは画面右上の☆マークをクリック、Edgeでは ★ アイコンまたは Ctrl+D でブックマークに追加できます。
2
「Log in」をクリックする
初回登録時と同じ方法（GoogleかEmail）でログインします。「Continue with Google」を選んだ方はGoogleアカウントを選択するだけで完了します。
3
チャット画面が開いたらOK
ブラウザが「ログイン状態を記憶」している場合、次回以降はclaudeを開くだけで自動的にチャット画面が表示されることもあります。
⚠️
会社のパソコンでは「ログイン状態を保持しない」設定になっている場合があります。毎回ログインが必要でも仕様ですので、ブックマークを活用してください。
🎊

インストール・アカウント作成・ログイン確認まで完了です。
